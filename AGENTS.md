# Repository Agent Instructions

## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for `LuisUrrutia/repository-finder`. See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the default triage labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Uses single-context domain docs: root `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.
